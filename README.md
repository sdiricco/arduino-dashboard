# arduino-dashboard

## Installation 

- Make sure you have Windows VS2017 pack installed. 

If have problems, try installing latest nodejs version, cocholatly tools and set VS2017 in environment

```sh
yarn
yarn electron-rebuild
```

## Start in dev mode

```sh
yarn dev
```

## Build 

- Make sure you have Windows VS2017 pack installed. 

If have problems, try installing latest nodejs version, cocholatly tools and set VS2017 in environment

```sh
yarn build
```
